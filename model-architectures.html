Parth Kalkar discusses a major optimization for Large Language Model (LLM) inference called YOCO (You Only Cache Once), which drastically reduces GPU memory usage by caching key-value pairs only once instead of at every layer. This approach splits the model into two parts—one that writes a single cache and another that reads from it—resulting in up to 80x lower memory footprint and 10x higher throughput compared to standard Transformers.
