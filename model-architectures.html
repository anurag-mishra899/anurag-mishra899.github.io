Parth Kalkar discusses a major memory optimization for large language model (LLM) inference called YOCO (You Only Cache Once). Instead of caching key-value pairs at every layer (as in standard Transformers), YOCO caches only once at a central layer, drastically reducing GPU memory usage and increasing throughput, making it feasible to run large context windows on consumer hardware.
