# AI Knowledge Base

Personal knowledge base for GenAI, LLMs, AI Agents, and ML Infrastructure.

## About

This site catalogs and organizes 101 articles, papers, tutorials, and resources about AI/ML topics collected from February to April 2026. Content is automatically extracted from URLs, summarized using Azure OpenAI, and organized into 8 main categories.

## Categories

- **LLM Training & Fine-tuning** (19 articles) - Model training, FSDP, LoRA, PEFT, DPO, GRPO, RLHF
- **LLM Inference & Serving** (15 articles) - vLLM, SGLang, KV cache, PagedAttention
- **AI Agents & Agentic Systems** (22 articles) - ReAct, harness engineering, MCP, memory systems
- **GPU Computing & Hardware** (13 articles) - CUDA, kernel optimization, H100, distributed training
- **RAG & Vector Search** (10 articles) - Vector databases, HNSW, Graph RAG, embeddings
- **Model Architectures** (12 articles) - Transformers, MoE, Mamba, vision-language models

- **Research & Learning** (6 articles) - Interview prep, LeetCode, courses, papers

## Features

- 📱 Responsive design with dark mode support
- 🔍 Client-side search functionality
- 🎥 YouTube video embeds
- 🏷️ Topic-based categorization
- 🔗 Related articles recommendations
- ⚡ Static site (no backend required)

## Technical Stack

- HTML5, CSS3 (responsive, dark mode)
- Vanilla JavaScript (search functionality)
- Generated with Python + Jinja2 templates
- Content summarized with Azure OpenAI GPT-4

## Source

This site is automatically generated from a knowledge base pipeline that:
1. Extracts URLs from WhatsApp chat exports
2. Fetches and parses content using BeautifulSoup
3. Generates summaries using Azure OpenAI
4. Creates raw markdown source files
5. Generates HTML pages with Jinja2 templates
