Alvaro Videla describes building a custom LLM inference engine that runs Qwen 2.5-0.5B entirely on A

Summary

Alvaro Videla describes building a custom LLM inference engine that runs Qwen 2.5-0.5B entirely on Apple’s Neural Engine (ANE), achieving 171 tokens/second with the GPU idle. By reshaping transformer operations as 1x1 convolutions, leveraging CoreML’s MLState API, and optimizing memory usage, he achieved a 19× speedup over the baseline, demonstrating efficient LLM inference on Apple Silicon.
