WhatSoup is a (deprecated) Python-based web scraper designed to export your entire WhatsApp chat history by automating a browser session, scraping cha

Summary

WhatSoup is a (deprecated) Python-based web scraper designed to export your entire WhatsApp chat history by automating a browser session, scraping chat messages (text only), and exporting them to .txt, .csv, or .html formats. It addresses WhatsApp's built-in export limitations but requires Chrome, ChromeDriver, and some Python setup, and does not download media files. The project is no longer maintained as of April 2021.
