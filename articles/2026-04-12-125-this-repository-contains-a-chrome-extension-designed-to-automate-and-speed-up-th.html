Summary

This repository contains a Chrome extension designed to automate and speed up the booking process for Shri Ram Janmbhoomi Darshan on the official portal. It auto-fills booking forms, saves user details, and provides quick access features, but requires manual captcha and OTP entry for security. All user data is stored locally and the extension does not bypass portal security measures.
