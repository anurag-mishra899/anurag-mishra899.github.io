Browser Harness is a self-improving framework that connects large language models (LLMs) directly to

📎 Original Source:

Summary

Browser Harness is a self-improving framework that connects large language models (LLMs) directly to a real Chrome browser using a thin, editable Chrome DevTools Protocol (CDP) harness. It enables LLMs to autonomously complete complex browser tasks, generate and refine helper code and reusable domain-specific skills, and supports community contributions for site-specific automations.
