Summary

The post explains when to choose pipeline parallelism over tensor parallelism for running large 70B parameter models, focusing on how GPU interconnects and network topology dictate the optimal parallelism strategy. It highlights that tensor parallelism is best within a single node with fast interconnects (like NVLink), while pipeline parallelism is preferable across multiple nodes with slower links (like PCIe or Ethernet). The recommended production approach is a hybrid of both, tailored to the hardware and network setup to maximize throughput and minimize latency.
