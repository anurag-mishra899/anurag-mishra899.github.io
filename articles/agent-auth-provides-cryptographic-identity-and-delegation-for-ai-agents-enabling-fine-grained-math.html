agent-auth provides cryptographic identity and delegation for AI agents, enabling fine-grained, mathematically enforced access control using Ed25519 s

GitHub Repository 📅 2026-03-17 👤 Shared by Anurag Mishra 🏷️ AI Agents & Agentic Systems
📎 Original Source:
https://github.com/kanoniv/agent-auth

Summary

agent-auth provides cryptographic identity and delegation for AI agents, enabling fine-grained, mathematically enforced access control using Ed25519 signatures and hierarchical scopes. It supports offline, verifiable delegation chains, audit trails, and integrates with tools like Claude Code and Git, with SDKs and CLIs in Python and Rust. The project emphasizes interoperability, with multiple independent agent systems cross-verifying delegation chains.
