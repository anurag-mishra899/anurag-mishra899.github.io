code-review-graph is a tool that builds a local, persistent knowledge graph of your codebase using T

GitHub Repository 📅 2026-04-08 👤 Shared by Anurag Mishra 🏷️ Uncategorized
📎 Original Source:
https://github.com/tirth8205/code-review-graph

Summary

code-review-graph is a tool that builds a local, persistent knowledge graph of your codebase using Tree-sitter, enabling AI coding assistants (like Claude Code, Copilot, etc.) to access only the relevant parts of your code during reviews and coding tasks. This approach significantly reduces token usage, speeds up context updates, and improves the accuracy of impact analysis, especially in large monorepos, by incrementally tracking code changes and dependencies.
