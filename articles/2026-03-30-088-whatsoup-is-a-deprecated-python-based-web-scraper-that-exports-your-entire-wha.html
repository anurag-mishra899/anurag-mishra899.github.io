Summary

WhatSoup is a (deprecated) Python-based web scraper that exports your entire WhatsApp chat history by automating a browser session, scraping chat messages, and exporting them to .txt, .csv, or .html formats. It addresses limitations of WhatsApp's native export feature but does not download media and requires Chrome, ChromeDriver, and some Python setup. The project is no longer maintained but can be forked and extended by others.
