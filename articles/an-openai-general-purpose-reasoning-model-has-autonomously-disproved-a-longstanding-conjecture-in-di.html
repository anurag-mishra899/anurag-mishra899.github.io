Summary

An OpenAI general-purpose reasoning model has autonomously disproved a longstanding conjecture in discrete geometry, specifically the planar unit distance problem posed by Paul Erdős. The model constructed an infinite family of point configurations that achieve a polynomial improvement over previously believed optimal constructions, using sophisticated tools from algebraic number theory. This marks a milestone in AI-driven mathematical research, demonstrating the model's ability to generate original proofs for prominent open problems.
