LiteParse is a fast, open-source, standalone document parser focused on local PDF and image parsing, offering spatial text extraction with bounding bo

GitHub Repository 📅 2026-03-20 👤 Shared by Sarvesh Tiwari EY 🏷️ Tools & Infrastructure
📎 Original Source:
https://github.com/run-llama/liteparse

Summary

LiteParse is a fast, open-source, standalone document parser focused on local PDF and image parsing, offering spatial text extraction with bounding boxes and built-in OCR using Tesseract.js. It supports multiple output formats, batch processing, screenshot generation, and can be used via CLI or as a library in Node.js projects, without requiring cloud dependencies.
