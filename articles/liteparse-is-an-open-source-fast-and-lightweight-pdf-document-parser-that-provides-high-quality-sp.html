LiteParse is an open-source, fast, and lightweight PDF document parser that provides high-quality spatial text extraction with bounding boxes, support

📎 Original Source:

Summary

LiteParse is an open-source, fast, and lightweight PDF document parser that provides high-quality spatial text extraction with bounding boxes, supporting both built-in and external OCR engines. It runs locally without cloud dependencies and offers multi-language support across Rust, Node.js/TypeScript, Python, and browser (WASM) environments, with a unified CLI and library API for parsing, batch processing, and screenshot generation.
