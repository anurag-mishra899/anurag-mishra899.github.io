📎 Original Source:

Summary

This repository demonstrates training neural networks directly on Apple's Neural Engine (ANE) by reverse-engineering private APIs, bypassing Apple's official inference-only restrictions. It provides proof-of-concept code, benchmarks, and documentation for running forward and backward passes—including transformer models—on ANE hardware, highlighting both the potential and current limitations of this approach. The project is intended as research, not a production framework.
