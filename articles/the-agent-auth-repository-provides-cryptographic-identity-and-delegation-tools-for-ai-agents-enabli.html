The agent-auth repository provides cryptographic identity and delegation tools for AI agents, enabling secure, scope-restricted actions using Ed25519

GitHub Repository 📅 2026-03-17 👤 Shared by Anurag Mishra 🏷️ AI Agents & Agentic Systems
📎 Original Source:
https://github.com/kanoniv/agent-auth

Summary

The agent-auth repository provides cryptographic identity and delegation tools for AI agents, enabling secure, scope-restricted actions using Ed25519 signatures and verifiable delegation chains. It offers Python and Rust SDKs, CLI utilities, and integrations for enforcing and auditing agent actions, including hierarchical scopes and cross-engine interoperability. The system ensures offline, third-party-free verification and detailed audit trails for agent activity.
