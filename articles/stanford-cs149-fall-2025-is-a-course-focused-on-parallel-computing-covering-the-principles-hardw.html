Summary

Stanford CS149 (Fall 2025) is a course focused on parallel computing, covering the principles, hardware, and software techniques necessary to design and program modern parallel systems. The course includes lectures on multi-core and GPU architectures, parallel programming models, optimization strategies, hardware specialization, and AI-driven performance, along with hands-on programming and written assignments.
