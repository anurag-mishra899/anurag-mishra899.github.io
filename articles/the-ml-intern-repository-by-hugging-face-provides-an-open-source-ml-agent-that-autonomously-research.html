The ml-intern repository by Hugging Face provides an open-source ML agent that autonomously researches, writes, and ships machine learning code using

Summary

The ml-intern repository by Hugging Face provides an open-source ML agent that autonomously researches, writes, and ships machine learning code using the Hugging Face ecosystem. It supports interactive and headless CLI modes, integrates with various LLM providers (Anthropic, OpenAI, local models), and offers features like session trace sharing, Slack notifications, and sandboxed tool execution for remote testing.
