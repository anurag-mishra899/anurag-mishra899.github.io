Summary

Gautam Ramachandra built a GPU compiler from scratch that compiles a C-like kernel language down to 16-bit binary instructions for an open-source Verilog GPU architecture, with the entire compilation and execution pipeline visible and interactive in a web browser. The project demonstrates real compiler techniques, hardware targeting, and provides live, cycle-accurate simulation, making the hardware-software boundary and execution process transparent for users.
