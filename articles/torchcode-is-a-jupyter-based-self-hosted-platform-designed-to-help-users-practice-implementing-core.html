TorchCode is a Jupyter-based, self-hosted platform designed to help users practice implementing core PyTorch operations and architectures from scratch

GitHub Repository 📅 2026-03-07 👤 Shared by Anurag Mishra 🏷️ Model Architectures
📎 Original Source:
https://github.com/duoan/TorchCode

Summary

TorchCode is a Jupyter-based, self-hosted platform designed to help users practice implementing core PyTorch operations and architectures from scratch, with instant auto-grading and feedback—similar to LeetCode but focused on tensors and machine learning interview topics. It features 40 curated problems covering fundamentals, attention mechanisms, architectures, optimization, and advanced techniques, with support for Docker, Google Colab, and Hugging Face Spaces.
